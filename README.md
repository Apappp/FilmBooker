
# FilmBooker

FilmBooker is a study project demonstrating clean software architecture, object-oriented programming, and strict adherence to the SOLID principles. The application provides a fully console-based interface for managing a cinema: movies, users, rooms, and seating layouts.


## Build instructions(Linux)

Clone the project

```bash
git clone https://github.com/Apappp/FilmBooker.git
cd FilmBooker
```
    
Create a build directory

```bash
mkdir build
cd build
```

Configure with CMake

```bash
cmake ..
```

Build

```bash
make
```

Run

```bash
./FilmBooker
```
